## ▶ PARAGRAPHS, HEADERS, BLOCKQUOTES  
### 1. Paragraph(단락)  
→ 하나 이상의 빈 줄로 구분된 연속된 텍스트 줄.  
→ 공백 or Tab만 포함된 줄은 공백으로 간주한다.  
`<p> This area is paragraphs.</p>`  <br>
`<p> 단락 내에서는 줄바꿈을 하려면 <br>를 사용해야한다. </p>` <br>
### 2. Headers(헤더)  
→ 헤더는 Setext, atx 두 가지의 Style이 있다.  
#### Setext Style  
`"<h1> Header </h1>"`  
`"Header"`  
`====`
→ h1을 나타냄.  

`"<h2> Header </h2>"`  
`"Header"`  
`----`
→ h2을 나타냄.
